## üöÄ Getting Started

### **Prerequisites**
- Node.js 18+ 
- npm or yarn package manager

### **Installation & Setup**

1. **Clone the repository**
```bash
git clone https://github.com/H3m0rv1ch1/portfolio.git
cd portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```

4. **Build for production**
```bash
npm run build
```

5. **Preview production build**
```bash
npm run preview
```
